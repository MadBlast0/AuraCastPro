# AuraCastPro AudioVirtualCable Kernel Driver
=============================================

## What this is
A Windows kernel-mode audio driver (WDM/PortCls) that creates:
  - "AuraCastPro Audio Output"  — render endpoint (AudioMixer writes here)
  - "AuraCastPro Audio Input"   — capture endpoint (OBS/Zoom/Teams read from here)

Audio is looped internally from output to input (virtual cable).
Format: 48000 Hz · 32-bit float · Stereo

## Build requirements
This driver CANNOT be built with CMake. It requires:
  - Visual Studio 2022 with WDK (Windows Driver Kit) installed
  - WDK version must match your Windows SDK version

### Install WDK:
  https://learn.microsoft.com/en-us/windows-hardware/drivers/download-the-wdk

## Build instructions
1. Open Visual Studio 2022
2. File → Open → Project/Solution → AudioVirtualCable.vcxproj
3. Select configuration: "Release" / "x64"
4. Build → Build Solution
5. Output: AudioVirtualCable.sys, Audio_Installer.inf, Audio_Bridge.cat

## Code signing
Development (test signing mode only):
  bcdedit /set testsigning on   [then reboot]
  See scripts/enable_test_signing.bat

Production:
  signtool sign /fd SHA256 /a /t http://timestamp.digicert.com AudioVirtualCable.sys

## Installation
The NSIS installer (scripts/package_installer.nsis) handles this automatically.
Manual install:
  pnputil /add-driver Audio_Installer.inf /install

## Files in this directory
