# Roboto Mono Font Files
Download from: https://fonts.google.com/specimen/Roboto+Mono
Required files:
  - RobotoMono-Regular.ttf
  - RobotoMono-Bold.ttf
  - RobotoMono-Medium.ttf

These are Apache 2.0 licensed fonts. Place downloaded .ttf files in this directory.
If missing at build time, the app falls back to the system monospace font automatically.
