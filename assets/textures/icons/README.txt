# App Icons
Generate from: assets/textures/icons/vector/app_icon.svg
Required files (in png/ subdirectory):
  - app_icon.png  (256x256, used in system tray and resources.qrc)
  - app_icon.ico  (multi-size ICO, used in AuraCastPro.rc)
  - ios.png       (48x48, iPhone device type indicator)
  - android.png   (48x48, Android device type indicator)

Use Inkscape: inkscape -w 256 -h 256 vector/app_icon.svg -o png/app_icon.png
